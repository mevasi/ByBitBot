## Setting up API keys

You will need a set of API keys from the Bybit exchange to use this bot

Please create a file named 'API', without the quotes, in the same directory as the trade.py file

Use the following format
* API
  ```sh
  THISISMYAPIKEY
  THISISMYAPISECRET
  ```
